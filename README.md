# Validação de expressões

Usando o conceito de pilha para fazer validação de expressões matemáticas.

## Iniciando

Digite a expressão e aperte "Enter" no seu teclado. O programa irá mostrar se a expressão é matematicamente válida ou não.

```
Exemplo de expressões válidas

-> {{[]}}  
-> {[]} 
-> {} 
-> [{}]

Exemplo de expressões inválidas

-> {[]}}  
-> {[} 
-> {}] 
-> [{}[}]

```
